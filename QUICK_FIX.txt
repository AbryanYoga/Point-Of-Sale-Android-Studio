🚨 LANGKAH WAJIB (URUTAN PENTING!):

1. SYNC GRADLE
   File → Sync Project with Gradle Files
   ⏳ Tunggu sampai selesai!

2. CLEAN PROJECT
   Build → Clean Project

3. REBUILD PROJECT
   Build → Rebuild Project

4. UNINSTALL APLIKASI LAMA
   Hapus aplikasi dari device/emulator

5. RUN APLIKASI
   Shift + F10

═══════════════════════════════════════════════════════════════

https://picsum.photos/400/300

🔍 CEK LOGCAT:

1. Buka Logcat (Alt+6)
2. Filter: tag:ModProduk
3. Lihat pesan error

═══════════════════════════════════════════════════════════════

✅ YANG BERHASIL:
- Toast: "Memuat gambar..."
- Gambar muncul
- Toast: "✓ Preview gambar berhasil dimuat"

❌ YANG GAGAL:
- Toast: "Gagal memuat gambar: [error]"
- Lihat detail error di Logcat
